AuroraLite – Serverless Task Management API

1.Overview
AuroraLite is a serverless REST API that allows users to create and retrieve tasks using AWS services.

2.Tech Stack
- AWS Lambda
- API Gateway
- DynamoDB
- Node.js

3.Features
- Create tasks (POST)
- Retrieve tasks (GET)
- Fully serverless and scalable

4.API Endpoints

POST:
https://k8rio9ihn9.execute-api.ap-south-1.amazonaws.com/Dev/tasks

GET:
https://k8rio9ihn9.execute-api.ap-south-1.amazonaws.com/Dev/tasks

5.Architecture
API Gateway → Lambda → DynamoDB
